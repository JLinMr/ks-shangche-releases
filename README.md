# 上车助手

快手直播 **小黄车定时上下车** 工具：在时间轴配置「上车」「下车」等节点，开播后按时间自动执行。

## 下载

👉 **[下载最新版（Windows 安装程序）](https://github.com/JLinMr/ks-shangche-releases/releases/latest)**

也可在 [Releases](https://github.com/JLinMr/ks-shangche-releases/releases) 页选择历史版本。已安装用户启动后，标题栏会提示新版本并引导下载。

## 系统要求

Windows 10 / 11（64 位）· 激活与快手功能需联网

## 使用说明

1. 安装后启动，输入**激活码**（见下方联系方式索取）。
2. 登录快手账号，在时间轴添加节点并**开始监控**。

## 常见问题

**无法激活 / 提示授权失效**

- 核对激活码是否输入正确、是否过期
- 确认网络正常，可访问授权服务
- 提示已绑定其他电脑：一码一机

**使用中突然提示未授权**

- 激活码可能被注销或到期，或长时间离线未通过联网校验，请重新激活或联系支持

**快手扫码登录失败 / 账号显示失效**

- 重新扫码登录；登录态过期较常见

**时间到了但没有自动上下车**

- 确认已点击**开始监控**，且时间轴上已添加对应节点。
- **上车、下车、讲解**等操作需账号**正在直播**，未开播无法执行。
- 确认节点绑定的商品 ID 正确，账号登录态有效。

**手动操作也失败（如加入货架、查询商品）**

- 先确认已选择账号且账号未失效
- 网络异常或快手接口限流时稍后重试

**更新后配置不见了**

- 若安装到新目录，旧数据仍在原安装路径，可将原目录下的 `data/` 与 `license` 复制到新安装目录。
